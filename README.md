# Installation
- Download [Bepinex](https://builds.bepinex.dev/projects/bepinex_be/577/BepInEx_UnityIL2CPP_x64_ec79ad0_6.0.0-be.577.zip).
- Open your game folder (In steam, right click placid plastick duck simulator, Manage > Browse local files).
- extract Bepinex so that all of it's files are in the game folder.
- run the game once. This will take some time.
- close the game.
- download `Splash.dll` from [Releases](https://github.com/o7Moon/Splash/releases).
- move `Splash.dll` to `(Game Folder from step 2)/BepInEx/plugins/Splash.dll`.

# Configuration
- there is a config file in your game folder at `BepInEx/config/PlacidPlasticDucks.Splash.cfg` with descriptions of all the available options.
